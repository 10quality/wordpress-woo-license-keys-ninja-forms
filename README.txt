Improve your customer support by integrating License Keys for WooCommerce plugin with Ninja Forms.

== Description ==

This plugin will integrate **License Keys** for **WooCommerce** with **Ninja Forms** by creating a *License Key* form field that will pull as dropdown the purchased license keys of a customer (with a search field). This field can be used to create better customer support forms and contact notifications.

== Form Field ==

**License Key** field that can be added to any form. The field will allow logged-in users to search and selected purchased license keys; guest users will have the option to type a license key instead.

== Merge Tag ==

If a license key field is part of the form. The merge tag **{licensekey:details}** can be used to further display license key information (customer code, product name, and order ID) to improve support notifications.

== Customization ==

This plugin has plenty of hooks for customization, all templates can also be edited inside a theme.

== Installation ==

1. Head to your Wordpress Admin Dashboard and click at the options **Plugins** then **Add new**.
2. Search for this plugin usign the search input or if have downloaded the zip file, upload it manually.
3. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Dependencies? =

Requires [License Keys for WooCommerce](https://wordpress.org/plugins/woo-license-keys/) and [Ninja Forms](https://wordpress.org/plugins/ninja-forms/) plugins to be installed and activated in your Wordpress setup.

= How to use the field? =

Once this plugin is activated, the "License Key" field will be available in the list of "User information fields".

= How to use the merge tag? =

Once this plugin is activated, a "License Key" merge tag group will be available in the list of merge tags.

For documentation [click here](https://www.10quality.com/docs/woocommerce-license-keys/).
